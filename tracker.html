AI Object Tracker

Raspberry Pi / CAD

Project Background

The goal of this project was to design and build an active AI object tracking system. This required integrating software-based computer vision models with physical hardware to actively track and follow targets in real-time.

My Role & Methodology

  • Programmed the core tracking logic and servo motor control using Python.
  • Trained and deployed the AI vision model using MediaPipe on a Raspberry Pi 5.
  • Designed and engineered a custom electromechanical camera holder from scratch using SOLIDWORKS.
  • Drafted the complete electrical schematics for the system.
